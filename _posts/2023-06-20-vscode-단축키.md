# 🎲 VS CODE 단축키 (맥북)

| 설명                         | 단축키 mac         |
| ---------------------------- | ------------------ |
| 자동 줄바꿈                  | option + z         |
| 원하는 줄로 이동             | ctrl + g           |
| 이 줄 바로 아래에 새 줄 입력 | cmd + enter        |
| 이 줄을 내리기               | option + down      |
| 이 줄을 올리기               | option + up        |
| 코드 자동 재정렬             | shift + option + f |
