원시(Primitive)타입
- string, number, boolean, null, undefined, biglnt, symbol

객체(Object)타입
-원시타입을 제외한 모든 것
object
array

변수는 어떠한 것을 저장할 상자다
원시 타입은 상자에 바로 들어간다
객체는 너무 크기 때문에 변수에 들어가는 게 아니라
메모리 상에 할당되어 그 메모리에 저장되고 메모리 주소가 상자에 들어간다


```javascript
const loactionOne = "korea"
const loactionTwo = "korea"

console.log(locationOne === locationTwo)

// > true

```

```Javascript
const locationOne = {
    country: "korea"
}
const locationTwo = {
    country: "korea"
}

console.log(locationOne === locationTwo)

// > false

```

변수라는 상자에는 메모리의 주소가 들어있고 다른 주소에 저장된 객체 두개는 같지 않게 되는 것이다.
